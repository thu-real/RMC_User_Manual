----------------------

-----------------------

（后缀为.30c），包括近400个核素，见 :numref:`acedata_table` 。此外，
发布包中还附带有部分热化数据
库，以及天然元素数据库（见 :numref:`natiso_table` ）。具体信息请参阅xdir文件。
